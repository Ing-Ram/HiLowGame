# HiLow Game
 Task: Write a GUI Hi-Lo guessing game using JavaFX without SceneBuilder.
 The program randomly picks a number from 1-100 (inclusive).
 The user enters a guess into a text field.
 The user can choose to press "enter" or click a button to submit.
 Both functionalities must be supported.
 Display a message about whether the guess was too low, too high, or correct.
 If the guess is wrong, allow the user to continue guessing.
 If the guess is right, allow the user to play again with a new number.
